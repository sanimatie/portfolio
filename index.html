Welkom op mijn portfolio!

In 2020 studeerde ik af als illustrator/animator met een interactieve geanimeerde website. Ik heb het maken van websites altijd al boeiend gevonden en merkte toen dat ik het volledig opbouwen uit code van mijn eigen website erg leuk vond. Omdat ik maar weinig van coderen afwist, was ik voornamelijk afhankelijk van wat ik op internet kon vinden. Daarna ben ik websites gaan bouwen met Wordpress en ik merkte dat ik eigenlijk het liefst code gebruikte om de front end in te richten, maar dat ik er simpelweg niet genoeg vanaf wist. Momenteel ben ik dan ook bezig mijn kennis uit te breiden en ben ik begonnen met het leren van HTML, CSS en JavaScript, met als doel deze kennis uiteindelijk in te zetten als front-end developer. De resultaten daarvan zie je op dit portfolio!
